import numpy as np
import random as rd

# Parámetros del problema
figus_total   = 860   # figuritas únicas del álbum Qatar 2022
figus_paquete = 5     # figuritas por paquete (sin repetición)
N             = 100   # repeticiones de la simulación

# Fijamos semilla para reproducibilidad
rd.seed(42)

# Muestra aleatoria: N repeticiones del experimento
muestras = [cuantos_paquetes(figus_total, figus_paquete) for _ in range(N)]

# Estimación de la esperanza por promedio muestral
estimacion_promedio = np.mean(muestras)

# Estadísticos descriptivos
estimacion_mediana = np.median(muestras)
estimacion_std     = np.std(muestras)
estimacion_min     = np.min(muestras)
estimacion_max     = np.max(muestras)
def crear_album(figus_total):
    """Crea un álbum vacío: vector de figus_total ceros."""
    return [0] * figus_total


def comprar_paquete(figus_total, figus_paquete):
    """Genera un paquete SIN repetición (garantía Panini)."""
    return rd.sample(range(0, figus_total), figus_paquete)


def pegar_figus(album, paquete):
    """Pega las figuritas del paquete en el álbum."""
    for figu in paquete:
        album[figu] = 1
    return album


def album_incompleto(album):
    """True si falta al menos una figurita."""
    return 0 in album


def cuantos_paquetes(figus_total, figus_paquete):
    """Simula llenado completo. Retorna cantidad de paquetes."""
    album              = crear_album(figus_total)
    paquetes_comprados = 0
    while album_incompleto(album):
        paquete = comprar_paquete(figus_total, figus_paquete)
        album   = pegar_figus(album, paquete)
        paquetes_comprados += 1
    return paquetes_comprados

Conclusiones

La distribución empírica presenta asimetría positiva (cola hacia la derecha): la mayoría completa el álbum cerca del valor esperado, pero existe probabilidad no despreciable de necesitar muchos más paquetes. Las últimas figuritas que faltan concentran la mayor dificultad, ya que la probabilidad de obtener una nueva decrece a medida que el álbum se llena.

Por la Ley de los Grandes Números (Maronna, Cap. 4), el promedio muestral converge al valor teórico a medida que N → ∞.
